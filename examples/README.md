# agently-edge Examples

This directory contains example agently-edge agent implementations.

## Available Examples

### hello-world

A minimal hello world agent demonstrating basic structure and configuration.

**Features**:
- YAML configuration loading
- Interactive conversation loop
- Mock LLM response

**Quick Start**:
```bash
cd examples/hello-world
cargo run
```

**Learn more**: [hello-world/README.md](hello-world/README.md)
