# agently Edge 产品定义

**版本**: 1.0  
**日期**: 2026-03-21  
**状态**: Draft

---

## 一、产品愿景

**让每一台商显设备都能运行 AI Agent**

---

## 二、硬件形态

### 1. 小屏广告机 (10-32 寸)

**部署场景**：
- 门店橱窗
- 收银台
- 电梯间
- 展柜

**典型用户**：顾客、店员

**核心价值**：把"广告牌"变成"销售员"

---

### 2. 户外中大屏商显 (43-98 寸)

**部署场景**：
- 商场中庭
- 楼宇外墙
- 广场大屏
- 展厅主屏

**典型用户**：路人、活动参与者

**核心价值**：把"展示屏"变成"互动场"

---

### 3. 会议一体机 (55-86 寸)

**部署场景**：
- 企业会议室
- 政府会议厅
- 教育培训室

**典型用户**：参会者、会议组织者

**核心价值**：把"显示设备"变成"会议助手"

---

## 三、价值维度

### 💬 人机互动

从"被动播放"到"主动交互"

**能力**：
- 语音交互
- 触摸交互
- 手势识别
- 主动问候

**技术栈**：
- 语音识别：Whisper-Tiny/Base
- 对话理解：Qwen2.5-1.5B-Q4
- 视觉感知：YOLOv8-Nano

---

### 🎨 内容生成

从"人工制作"到"AI 生成"

**能力**：
- 文案自动生成
- 多语言自动适配
- 实时转写 + 摘要
- 千人千面推荐

**技术栈**：
- 文案生成：Qwen2.5-1.5B
- 摘要生成：同模型复用
- 翻译：NLLB-200

---

### 📁 内容管理

从"手动更新"到"智能投放"

**能力**：
- 云端统一下发
- 分时段/分人群投放
- 多屏协同播放
- 语义检索

**技术栈**：
- 向量数据库：ChromaDB/FAISS
- 内容审核：本地分类模型
- agently 协议

---

### 🔧 设备管理

从"被动维修"到"预测维护"

**能力**：
- 远程状态监控
- 故障自动告警
- 批量固件升级
- 预测性维护

**技术栈**：
- 设备监控 Agent
- 预测模型
- IoT 传感器

---

## 四、技术规格

### 目标设备 (L2 主流)

| 配置项 | 规格 |
|--------|------|
| CPU | 8 核 ARM Cortex-A73 |
| 内存 | 2GB |
| 存储 | 16GB |
| 系统 | Android 10+ / Linux / 鸿蒙 |

### 内存占用

| 模块 | 内存 |
|------|------|
| 核心运行时 | 200MB |
| 对话模型 | 1GB |
| 语音识别 | 100MB |
| 语音合成 | 100MB |
| 向量数据库 | 200MB |
| 设备管理 | 100MB |
| 系统预留 | 300MB |
| **总计** | **~2GB** |

### 推理引擎

- **主引擎**: llama.cpp
- **模型格式**: GGUF
- **量化支持**: Q4_K_M / Q5_K_M / Q8_0
- **推荐模型**: Qwen2.5-1.5B-Instruct-GGUF

## 六、商业模式

### 收入来源

| 来源 | 说明 | 定价 |
|------|------|------|
| 设备授权 | 硬件厂商预装 | ¥50-200/台 |
| 云端服务 | 设备管理、模型调用 | ¥100-500/台/年 |
| 应用分成 | 通过平台分发的应用 | 15-30% |
| 企业支持 | 定制开发、技术支持 | 项目制 |

### 飞轮效应

```
更多硬件厂商预装
    ↓
设备基数大
    ↓
开发者愿意开发应用
    ↓
应用生态丰富
    ↓
更多硬件厂商想预装
    ↓
循环加速 → 壁垒形成
```

## 七、竞争分析

### 竞品对比

| 维度 | Edge Impulse | AWS Greengrass | agently Edge |
|------|-------------|----------------|-------------|
| 目标设备 | 单片机/嵌入式 | IoT 网关 | Android/Linux 终端 |
| 核心能力 | 模型推理 | 云端协同 | 完整 Agent 运行时 |
| 开发语言 | C++/Python | AWS SDK | Python/JavaScript |
| 应用场景 | 工业/传感器 | IoT 自动化 | 商业显示/交互 |
| 开源 | 部分开源 | 闭源 | 完全开源 |

### 差异化定位

- **Edge Impulse**: 嵌入式 ML 推理
- **AWS Greengrass**: IoT 云端协同
- **agently Edge**: 智能终端 Agent 平台 ← 空白市场
