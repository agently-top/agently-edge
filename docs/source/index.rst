产品概述
========

agently Edge 是一个运行在边缘设备上的 AI Agent 运行时平台，专为商显设备设计：

- 📱 **小屏广告机** (10-32 寸) — 门店橱窗、收银台、电梯间
- 🖥️ **户外大屏商显** (43-98 寸) — 商场中庭、楼宇外墙、广场大屏
- 🎤 **会议一体机** (55-86 寸) — 企业会议室、政府会议厅、培训室

核心特性
--------

- **跨平台**: Linux、Android、鸿蒙 Next 完整支持
- **高性能**: Rust 核心运行时，内存占用 <50MB
- **离线运行**: 不依赖云端，数据不出设备
- **易扩展**: 统一的 Agent 开发接口
